# ai_verify_frontend
# Development Mode
## Version
1.0.0

## Overview
This is the frontend of the AI Verify platform, built using React, TypeScript, and Vite. It includes role-based user management, dashboard functionality, and API integration for document validation and collaboration.

## Prerequisites
- Node.js >= 18.x
- npm >= 9.x
- Modern browser (Chrome, Firefox, Edge)

## Project Structure
- `src/`: Source code
- `public/`: Static files
- `.env.*`: Environment-specific variables
- `vite.config.ts`: Vite configuration

## Setup Instructions

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
